## 📌 Project Overview

This project was developed to simulate real-world airline analytics and strategic decision-making for IndiGo Airlines.

The objective is to answer critical business questions such as:

* Which routes generate the highest profit?
* Which airports serve as strategic network hubs?
* Which routes should be expanded or optimized?
* How will passenger demand evolve over time?
* What is the impact of fuel costs, ticket pricing, and load factor changes on profitability?
* Which airports should be prioritized for future network expansion?

---

## 🎯 Business Objectives

* Analyze IndiGo's route profitability.
* Identify high-performing and underperforming routes.
* Discover critical hub airports using network analysis.
* Forecast passenger demand and revenue.
* Simulate strategic business scenarios.
* Provide actionable recommendations for route expansion.
